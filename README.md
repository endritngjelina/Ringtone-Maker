# 🎶 Ringtone Maker

A browser-based ringtone creation tool built entirely with HTML, CSS, and JavaScript. Compose, customize, and export your own ringtones — no software installation required.

***

## 🎵 Overview

Ringtone Maker is a series of 7 progressively developed HTML5 apps (`Ringtone Maker 0.html` → `Ringtone Maker 6.html`) that allow users to create custom ringtones directly in the browser. Each version builds upon the last, introducing new features, UI improvements, and audio capabilities using the Web Audio API.

***

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- Speakers or headphones to preview your ringtone
- *(Optional)* Microphone access for sound-input features

### How to Use

1. Clone or download this repository:
   ```bash
   git clone https://github.com/endritngjelina/Ringtone-Maker.git
   ```
2. Open the latest version in your browser for the best experience:
   ```
   Ringtone Maker 6.html
   ```
3. Use the on-screen controls to compose your ringtone.
4. Preview your creation and export/download the audio file.

> **Tip:** Open `Ringtone Maker 0.html` to trace the project's evolution from prototype to polished app.

## ✨ Features

- 🎹 Compose custom melodies using selectable notes and beats
- ▶️ Real-time audio preview directly in the browser
- 🎚️ Adjust tempo, pitch, and tone settings
- 💾 Export/download your ringtone as an audio file
- 📱 Lightweight and mobile-friendly — no frameworks or dependencies
